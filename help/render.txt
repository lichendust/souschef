Render will start the job queue, rendering each job sequentially.  If any job fails, at any stage, it warn the failure and progress to the next one in the queue.

$1Usage$0
-----

    souschef $1render$0 [--watch -w]

$1Watch$0
-------------

    $1--watch -w$0

Enabling "watch" mode will cause Sous Chef to remain alive even if there are no available jobs, watching the jobs directory for new additions.  In this mode Sous Chef *will* automatically delete finished jobs and any associated cache data to save space.
